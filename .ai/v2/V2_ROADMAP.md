# V2 Roadmap

## Phase 1: Domain and data foundation
- migrations
- ORM models
- audit events
- repository layer

## Phase 2: Recommendation and follow-up
- recommendation service
- follow-up trigger service
- follow-up task APIs
- workbench panels

## Phase 3: Customer tags and profile
- tag rules
- profile snapshot service
- profile UI integration

## Phase 4: Operation center
- audience filter
- campaign/task generation
- manual confirmation flow

## Phase 5: Analytics and lightweight risk
- analytics aggregation jobs
- dashboards
- risk flag rules
- risk badges/reminders

## Delivery priority
1. recommendation center
2. follow-up task center
3. customer tags and profile snapshot
4. operation task center
5. analytics dashboard
6. lightweight risk flags

## Explicit non-goals for V2
- full quality inspection center
- full risk center
- ERP / OMS / WMS deep integration center
- VOC analysis center
- training center
- full commercial delivery system
